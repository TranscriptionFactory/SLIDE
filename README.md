```install_github("jishnu-lab/SLIDE")```

```library(doParallel)```


```SLIDE(z,y,method = 4,do_interacts = TRUE,betas = NULL,spec = 0.3,fdr = 0.1,niter = 1000,elbow = FALSE,f_size = 100,parallel = TRUE,ncore = 10)```
